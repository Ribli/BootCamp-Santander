Tipos

Primitivos

numbers

São numeros, podem ser utilizados em cálculos arítmeticos.

strings

São textos, declarados entre aspas ou crase, e quando é executado como no exemplo, se torna um objeto.
Exemplo: let exemplo = new String("blablabla") e utilizando exemplo."metodos que podem ser utilizados".

boolean

Verdadeiro "true" ou falso "false" uma resposta referente a validação.

arrays

São listas iteráveis de elementos. let array = [1, 2, 3, 2].
array.push(3) método para incluir itens no final da lista array.

Objetos

Estrutura do tipo chave valor,
Exemplo: let chave = { nome:'Ribli', idade: 31, natural:'Ilhéus'};

Empty, null e undefined

Empty = declarou a variável ou array, number = 0, string = "" não tem valor dentro
Null = quer que o valor não exista, seja falso nulo, let falso = null
undefined = nem foi declarado, não existe var ou algo com está definição
